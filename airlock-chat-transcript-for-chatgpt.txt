PRODUCT CONTEXT (from README)
================================================================================

Airlock: local-first disposable sandboxed workspace.
- User drops file → ephemeral Docker container with KasmVNC desktop
- File opened in sandboxed app (Chromium, Evince, etc.)
- VNC streamed to Electron window
- Container destroyed on session end
- Air-gapped by default (v0.2.0 for full egress block)
- Violent GC on Electron crash (docker kill all session containers)
- Read-only file mounts

Monorepo:
  packages/ui     — Electron + React
  packages/core   — dockerode orchestrator
  packages/sandbox — Docker image (Alpine + KasmVNC + Openbox)
